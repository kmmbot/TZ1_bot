Для корректной работы бота перед его использованием следует изменить некоторые данные в config.cfg.
Напрмиер, поля url_search, url_token, url_payment и url_subs заменить на адрес вашей ВМ.
В поле secret необходимо указать код для шифрования данных при запросе к биллингу. Он указан в настрйоках API непосредственно в биллинге в разделе "внешние интеграции".
Кроме этого, для доступе к БД (раздел sql  в файле) следует создать нового пользователя в базе данных и выдать ему права на чтение таблицы packets и указать логин и пароль для этого 
пользователя в поля user и password в конфиг файле. Также необходимо изменить адрес host на актуальный адрес БД.

Остальные поля менять необзодимости на данном этапе работы нет. Поле api - ключ для СМС рассылки, которая для удобства тестирования программы, сейчас отключена. Код для входа в ЛК 
в боте можно найти в стандартном потоке вывода PyCharm. Изменить это можно отредактировав строки 301-315 в функции message_action программы bot_itself.
